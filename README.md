# Lights
### You can create your custom light anywhere.
### Вы можете создать свой кастомный свет где-угодно.
## Config
```
- Light_Name: Any_convenient_name
  Room: Room_name
  Color: Color_name
  Position:
    x: 0
    y: 0
    z: 0
  Intensive: 1
  Range: 10
```
